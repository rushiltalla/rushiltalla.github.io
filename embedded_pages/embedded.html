Embedded Software

Embedded software development using various languages and skills.
Skills: C++, Python, Communication (UART), Soldering, 3D Modeling

About

Here are some embedded software development projects I have worked on and improved my skills with.
These helped me develop many skills in programming and resourcefulness (using online resources like
data sheets to learn and use technology) and explore electrical/embedded engineering through working with
circuits and soldering.

Arduino Thermometer

A medal winning thermometer for Science Olympiad using a thermister
at the tip of a 3D printed probe. Arduino measures changes in voltage
and uses a mapping function to generate a temperature reading.
Skills: C++, Soldering, 3D Modeling, Data Collection/Analysis

UART IMU for Beaglebone

A project for the Michigan Mars Rover team that consisted of writing
code to read data from an IMU using UART and export data to a struct
containing magnetometer and other positioning data.
Skills: Python, Beaglebone wiring, UART serial interface

Card Dealing Robot

A small robot capable of dealing cards around it by using a spinning
wheel to eject cards. Required using two motors (rotating and ejecting)
along with buttons to let users set number of players and cards per player.
Skills: C++, Soldering, 3D Modeling, Arduino wiring (with breadboard)
